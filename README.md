flexiforce-arduino-multitech
============================

Code written for the Axeda M2M Hackathon in Boston, May 2013

Integrates FlexiForce sensors to A-D pins, filters the data according to threshholds (soft and hard keypresses), and outputs PIN strings.

Also includes code to send the PIN up to the Axeda cloud via the MultiTech OCG

For more information, see this post:
http://wkchen.tumblr.com/post/49814290645/pushpin-next-generation-lock-technology
